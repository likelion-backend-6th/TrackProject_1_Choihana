# TrackProject_2_Choihana
## 도서 서비스 프로젝트 📚 

### 개요
- 프로젝트 기간: 2023.09~
- 주요 기능: 가입/로그인, 도서 조회, 대여/반납, 리뷰 작성
- 멤버: 개인 프로젝트
